messageHandlerState_t Enum

Header: #include <modbus.h>

The current state/action of the Modbus message handler.


    typedef enum
    {
        success,
        failure,
        waiting
    } messageHandlerState_t;
Value Description
success A complete Modbus message has been received and validated.
failure Either the Modbus message has timed out or it is invalid.
waiting The device is still waiting for a Modbus response.
