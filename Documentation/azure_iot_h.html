Modbus azure_iot.h

Header: #include <azure_iot.h>

The Modbus azure_iot header contains functions that allow the Azure Sphere to communicate with the IoT Hub.

Functions

Function Description
AzureIoT_EventHandler Primary handler to deal with Azure IoT Hub connectivity and activities.
AzureIoT_SendMessage Sends telemetry data to IoT Hub
AzureIoT_TwinReportState Enqueues a report.
AzureIoT_AddTwinUpdateCallback Add a callback to the list to be searched when the Twin is updated.

Typedefs

Typedef Description
TWIN_CALLBACK The trigger and the function to call when the trigger is reached for a twin callback.
TwinUpdateContext Values to be passed back to the callback function.
