WriteFile Function

Header: #include <modbus.h>

Sends a request to write to a file stored on the slave device.


    bool WriteFile(modbus_t hndl, uint8_t slaveID, uint8_t* messageArray, uint8_t messageLength, uint8_t* readArray, size_t timeout);

Parameters

  • hndlThe message handle.

  • slaveIDAddress of the slave device.

  • messageArrayThe array that the write subrequests will be stored in.

  • messageLengthThe length of the array of subrequests.

  • readArrayPointer to an array to store the slave device's confirmation message.

  • timeoutTime in milliseconds after which function will return an error if no response has been received from the device.

Returns

true on success, or false on failure
