ModbusConnectRtu Function

Header: #include <modbus.h>

Creates and set up a socket for serial data, and returns an opaque handle to be used by the Modbus low level APIs.


    modbus_t ModbusConnectRtu( serialSetup setup, size_t timeout );

Parameters

  • setupThe serial parameters that the M4 will use to transmit and recieve data.

Returns

Modbus handle on success, or null on failure
