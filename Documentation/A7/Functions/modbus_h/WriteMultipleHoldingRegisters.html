WriteMultipleHoldingRegisters Function

Header: #include <modbus.h>

Sends a request to write to a variable number of holding registers.


    bool WriteMultipleHoldingRegisters( modbus_t hndl, uint8_t slaveID, uint16_t address, uint16_t numToWrite, uint16_t *registerArray, uint8_t* readArray, size_t timeout );

Parameters

  • hndlThe message handle.

  • slaveIDAddress of the slave device.

  • addressAddress of the first register to write to.

  • numToWriteNumber of registers to write.

  • registerArrayPointer to the array of data to write from.

  • readArrayPointer to an array to store the error code response if present.

  • timeoutTime in milliseconds after which function will return an error if no response has been received from the device.

Returns

true on success, or false on failure
