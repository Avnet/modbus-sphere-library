Azure Sphere Modbus

The Azure Sphere modbus library supports communication with various modbus devices across TCP, RTU and RTU/TCP.

Functions

Header Description
modbus.h The core modbus functions to be run on the A7
azure_iot.h Communicates with the azure IoT Hub to enable telemetry, twin updates and control of the device via the IoT Hub.
message-handler.h Runs on the M4 to handle RTU (RS485) transmissions coming in and out of the A7.
modbusCommon.h Definitions and enums that are shared and used by both libraries for A7 and M4.
