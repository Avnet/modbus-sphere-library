ModbusConnectTcp Function

Header: #include <modbus.h>

Creates and sets up a socket for TCP, and returns an opaque handle to be used by the Modbus low level APIs.


    modbus_t ModbusConnectTcp( const char* ip, uint16_t port );

Parameters

  • ipThe IP address of the device to be connected to.

  • portThe port of the device to be connected to.

Returns

Modbus handle on success, or null on failure
