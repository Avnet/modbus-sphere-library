SetMessageData Function

Header: #include <modbus.h>

Set the message content for an outgoing message.


    bool SetMessageData(messageHandle *message, uint8_t *body, size_t length);

Parameters

  • messageThe message to be written to.

  • bodyA pointer to the beginning of the data to be set.

  • lengthThe length of the data to be set.

Returns

true on success, or false on failure
