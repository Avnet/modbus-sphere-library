ReadA7Message Function

Header: #include <modbus.h>

Checks to see if a message has arrived from the A7, if so it stores it in message for future use.


    bool ReadA7Message(BufferHeader *inbound, BufferHeader *outbound, uint32_t sharedBufSize, messageHandle *message);

Parameters

  • inboundA buffer header containing the read and write positions of incoming data.

  • outboundA buffer header containing the read and write positions of outgoing data.

  • sharedBufSizeThe size of the shared buffer between the A7 and M4 that is used for intercore communications.

  • messageThe handle for the message to be stored in.

Returns

true if a message has been read, or false if one has not
