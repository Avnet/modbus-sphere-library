SendA7Message Function

Header: #include <modbus.h>

A function to send the data stored in the messagehandle to the A7.


    void SendA7Message(BufferHeader *inbound, BufferHeader *outbound, uint32_t sharedBufSize, messageHandle *message);

Parameters

  • inboundA buffer header containing the read and write positions of the receive buffer.

  • outboundA buffer header containing the read and write positions of the sending buffer.

  • sharedBufSizeThe size of the shared buffer between the A7 and M4 that is used for intercore communications.

  • messageThe message to be sent.
