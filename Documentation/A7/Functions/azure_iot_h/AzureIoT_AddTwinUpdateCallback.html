AzureIoT_AddTwinUpdateCallback Function

Header: #include <modbus.h>

Add a callback to the list to be searched when the Twin is updated.


    bool AzureIoT_AddTwinUpdateCallback(const char* property, twinCallback fPtr, TwinUpdateContext context);

Parameters

  • propertyA string containing the desired property change that will trigger the callback.

  • fPtrThe callback to use<.

  • contextParameters to be passed to the callback.

Returns

true if added, false if not enough space to store it
