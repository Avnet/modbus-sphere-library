serialPortMsgTypes Enum

Header: #include <modbus.h>

The type of serial port message being transmitted.


    {
        UART_CFG_MESSAGE = 1,
        UART_CFG_MESSAGE_RESPONSE,
    } serialPortMsgTypes;
Value Description
UART_CFG_MESSAGE The transmission being sent is a configuration for the devices header pins.
UART_CFG_MESSAGE_RESPONSE The transmission being sent is a response to a UART_CFG_MESSAGE.
