Modbus message-handler.h

Header: #include <message-handler.h>

The Modbus message-handler header contains functions that support tranmission between the A7 and M4 in order to send and receive data to the physical header pins.

Functions

Function Description
ReadA7Message Checks to see if a message has arrived from the A7, if so it stores it for future use.
GetMessageProtocol A function to retrieve the protocol byte from the message.
GetMessageCommand A function to retrieve the command byte from the message.
GetMessageLength A function to retrieve the length of the message.
GetMessageMaxLength A function to retrieve the maximum message length of a message.
GetMessageDataPtr A function to retriveve the pointer to the beginning of the message
GetMessagePrefixPtr A function to retrieve the pointer to the message prefix (for intercore communications)
SetMessageProtocol Set the protocol byte for an outbound message
SetMessageCommand Set the command byte for an outbound message
SetMessagePrefix Set the message prefix for an outgoing message (for intercore communications)
SetMessageData Set the message content for an outgoing message.
SetMessageLength Set the message length for an outgoing message.
SendA7Message A function to send the data stored in the messagehandle to the A7.
InitMessage Initialize a messagehandle.

Typedefs

Typedef Description
messageHandle Stores modbus messages to and from the A7 as well as the important indicies within them.
