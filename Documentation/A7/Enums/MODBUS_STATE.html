MODBUS_STATE Enum

Header: #include <modbus.h>

The current state/action that the Modbus instance is performing.


    typedef enum
    {
        Idle,
        SendingRequest,
        WaitingForResponse,
        DataReceived,
        TransactionFailed
    } MODBUS_STATE;
Value Description
Idle The Modbus instance is waiting to receive a task.
SendingRequest The Modbus instance is sending a request to its target device.
WaitingForResponse The Modbus instance has sent a request and is waiting for a response.
DataReceived The target device has successfully responded to the device.
TransactionFailed The target device has either taken too long or has not sent the expected response.
