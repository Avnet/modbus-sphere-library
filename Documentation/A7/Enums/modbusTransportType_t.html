modbusTransportType_t Enum

Header: #include <modbus.h>

The type of transport Modbus uses to transmit its data.


    typedef enum
    {
        tcp,
        rtuOverTcp,
        rtu
    } modbusTransportType_t;
Value Description
tcp The Modbus instance will be using Modbus TCP
rtuOverTcp The Modbus instance will be using Modbus RTU/TCP
rtu The Modbus instance will be using Modbus RTU
