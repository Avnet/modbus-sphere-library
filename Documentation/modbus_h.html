Modbus modbus.h

Header: #include <modbus.h>

The modbus header contains functions that allow the Azure Sphere to communicate to other modbus compatible devices.

Functions

Function Description
ModbusInit Initialises the Epoll thread and sets up the relevant variables.
ModbusExit Closes the Epoll thread and cleans up relevant variables.
ModbusErrorToString Takes error code from a modbus message and returns the error message as a string.
ModbusConnectTcp Creates and sets up a socket for TCP, and returns a message handle with all of the relevant information.
ModbusConnectRtuOverTcp Creates and sets up a socket for RTU over TCP, and returns a message handle with all of the relevant information.
ModbusConnectRtu Creates and set up a socket for serial data, and returns a message handle with all of the relevant information.
ModbusClose Closes the connection created by ModbusConnectIp/ModbusConnectRtu and frees the memory taken up by the handle.
ReadCoils Sends a request to read a variable number of coils.
ReadDiscreteInputs Sends a request to read a variable number of discrete inputs.
ReadMultipleHoldingRegisters Sends a request to read a variable number of holding registers.
ReadInputRegisters Sends a request to read a variable number of input registers.
ReadFile Sends a request to read from a file stored on the slave device.
ReadFileSubRequestBuilder Creates and appends (if there is already a subrequest) a new subrequest to be placed into the ReadFile request message.
WriteSingleCoil Sends a request to write to a single coil.
WriteSingleHoldingRegister Sends a request to write to a holding register.
WriteMultipleCoils Sends a request to write to a variable number of coils.
WriteMultipleHoldingRegisters Sends a request to write to a variable number of holding registers.
WriteFile Sends a request to write to a file stored on the slave device.
WriteFileSubRequestBuilder Creates and appends (if there is already a subrequest) a new subrequest to be placed into the WriteFile request message.
