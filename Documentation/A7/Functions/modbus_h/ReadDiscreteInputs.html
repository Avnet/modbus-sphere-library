ReadDiscreteInputs Function

Header: #include <modbus.h>

Sends a request to read a variable number of discrete inputs.


    bool ReadDiscreteInputs( modbus_t hndl, uint8_t slaveID, uint16_t address, uint16_t bitsToRead, uint8_t* readArray, size_t timeout );

Parameters

  • hndlThe message handle.

  • slaveIDAddress of the slave device.

  • addressAddress of the first discrete input to read on the device.

  • bitsToReadNumber of discrete input to read.

  • readArrayPointer to an array to store read data in.

  • timeoutTime in milliseconds after which function will return an error if no response has been received from the device.

Returns

true on success, or false on failure.
